三、进阶配置详解

配置文件管理（核心部分）

  1. 订阅链接导入

    • 支持Clash、Surge等格式的订阅
    • 推荐使用Base64编码增强安全性

  2. 手动配置示例
    ```yaml
    proxies:

    • name: "JP-01"
      type: vmess
      server: jp.example.com
      port: 443
      uuid: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
      alterId: 0
      cipher: auto
      tls: true
      ```

  3. 规则集配置技巧

    • 利用GEOSITE/GeoIP数据库实现智能分流
